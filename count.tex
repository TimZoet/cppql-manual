To count the number rows in a table, you can construct a \gls{count_class} object from a \gls{typed_table_class}. This object holds a statement of the form \code{SELECT COUNT(*) FROM <table> WHERE <expr>}. See Listing~\ref{lst:count:create}.

\lstinputlisting[caption={Creating a count statement.}, label={lst:count:create}]{snippets/count_create.cpp}

Calling the count object will directly return the number of rows that match the filter expression. Optionally, you can pass a boolean parameter to indicate whether filter parameters should be rebound. See Listing~\ref{lst:count:call}.

\lstinputlisting[caption={Counting rows.}, label={lst:count:call}]{snippets/count_call.cpp}

The \code{countAll} method of the \gls{typed_table_class} can be used to count all rows in a table, without a filter expression.
